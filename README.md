Marcus: Santo Domingo Street Justice

Reh Dogg Games Plus presents an open-world action game prototype set in a fictional Caribbean city inspired by Santo Domingo.
Players take the role of a vigilante working to protect neighborhoods, complete missions, and restore order across the city.

This project demonstrates:

Mission-based gameplay
Video cutscenes for success and failure
Intro and start menu flow
Third-person movement and driving
Radar / HUD system
Modular video integration
Browser-based 3D gameplay using Three.js
